# Macro App

A single-file calorie and macro calculator (HTML/CSS/JS, no dependencies).

## Features
- Set a total daily calorie baseline
- Split into protein / carbs / fat with sliders or % inputs (default 50 / 25 / 25)
- Percentages auto-rebalance to 100%
- Quick presets (high protein, balanced, lean bulk, keto, endurance)
- Food log: enter raw-weight grams of foods (white rice, chicken breast, egg white, etc.) and compare against your targets

## Usage
Open `index.html` in any browser. Works offline.

Macro energy: protein and carbs 4 kcal/g, fat 9 kcal/g. Food values are typical USDA raw figures and vary by product.
